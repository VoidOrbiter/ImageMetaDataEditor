Image MetaData Editor
---------------------
This software is a work in progress, though it shouldn't take much longer for it's initial version release as it's really a simple application. 

WHAT DOES THIS SOFTWARE DO?
---------------------------
This software will allow the editing of Image MetaData for any use case. The reason for creating this is simply due to a personal interest in ARGs. Though, anyone can use this for any reason I suppose.
Currently, this software can only display .jpg MetaData, eventually it will display .png, .jpeg, .webp, and more image file types as time goes on.
